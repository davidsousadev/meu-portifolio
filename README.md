# meu-portfolio

Meu portfólio pessoal.

## Sobre

Este repositório contém o site do meu portfólio pessoal, com páginas sobre mim, formação, experiências, projetos e contatos. O objetivo é demonstrar habilidades em HTML, CSS e JavaScript através de páginas estáticas e pequenos projetos interativos.

## Pré-visualização

Você pode visualizar o site abrindo o arquivo `index.html` no navegador ou hospedando o diretório em um servidor estático. Uma prévia das imagens usadas está na pasta `imagens/`.

## Páginas principais

- `index.html` — página inicial
- `sobre.html` — sobre / biografia
- `portifolio.html` — projetos e exemplos
- `contatos.html` (em `li/contatos.html`) — informações de contato
- Outras páginas: `habilidades.html`, `formacao.html`, `experiencias.html`

## Projetos incluídos

Na pasta `projetos/` há várias demos e jogos (por exemplo, jogos/mario, jogo-da-memoria, piano), cada uma com seus próprios arquivos estáticos e assets.

## Tecnologias

- HTML5
- CSS3 (inclui arquivos em `assets/css/`)
- JavaScript (arquivos em `assets/js/` e `projetos/*/src/scripts`)
